ENTRENAMIENTO PERSONAL

IBM Cloud Essentials

IBM Cloud es una plataforma en la nube de estándares abiertos para crear, ejecutar y gestionar aplicaciones. Con IBM Cloud, los desarrolladores pueden centrarse en crear experiencias de usuario excelentes con opciones de computación flexibles, la opción de herramientas DevOps y un potente conjunto de API y servicios de IBM y de terceros. Este curso te ayuda a comprender los fundamentos de cloud-computing, IBM Cloud, y las aplicaciones Cloud Foundry.

Watson & Chatbots

En este curso, crea tu propio chatbot utilizando el servicio de conversación de IBM Watson. Tu chatbot simulará una conversación en torno a algunos de los parques nacionales más interesantes de EE.UU., Como Arches y Yellowstone. Aprenderas a entrenar a su bot, crear su diálogo y escribir código para integrarse con API externas. Con el servicio Conversation de IBM Watson ™, puedes crear una aplicación que entienda las entradas de lenguaje natural y use el aprendizaje automático para responder a los usuarios de una manera que simule una conversación entre humanos.

Cloud Application

Este curso proporciona una revisión de conceptos clave para ayudarlo a prepararse para el de desarrollo de aplicaciones IBM Cloud Platform. Está estructurado como una serie de videoconferencias acompañadas de demostraciones, laboratorios prácticos y cuestionarios. Completar este curso le dará experiencia con IBM Bluemix y le mostrará cómo crear aplicaciones para la nube que sean escalables, confiables y seguras.

Blockchain Essentials

Blockchain es una tecnología emergente que puede mejorar radicalmente la banca, la cadena de suministro y otras redes de transacciones y puede crear nuevas oportunidades para la innovación. Las empresas contienen muchos ejemplos de redes de personas y organizaciones que colaboran para crear valor y riqueza. Estas redes trabajan juntas en mercados que intercambian activos en forma de bienes y servicios entre los participantes. Este curso te ayudará a aprender acerca de blockchain para empresas y explorar casos de uso clave que demuestran cómo la tecnología agrega valor.
