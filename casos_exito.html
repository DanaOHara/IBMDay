IBM Cloud Essentials

IBM Cloud es una plataforma en la nube de estándares abiertos para crear, ejecutar y gestionar aplicaciones. Con IBM Cloud, los desarrolladores pueden centrarse en crear experiencias de usuario excelentes con opciones de computación flexibles, la opción de herramientas DevOps y un potente conjunto de API y servicios de IBM y de terceros. Este curso te ayuda a comprender los fundamentos de cloud-computing, IBM Cloud, y las aplicaciones Cloud Foundry.
